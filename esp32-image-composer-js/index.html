Flashing Commands

Use these commands with esptool.py to flash the complete firmware image:

Flash Individual Partitions
python -m esptool --chip esp32p4 -b 460800 --before default_reset --after hard_reset write_flash --flash_mode dio --flash_size detect --flash_freq 80m
Flash Complete Downloaded Image
python -m esptool --chip esp32p4 -b 460800 --before default_reset --after hard_reset write_flash --flash_mode dio --flash_size detect --flash_freq 80m 0x0 esp32-p4-complete-firmware-.bin

Replace <timestamp> with the actual timestamp from your downloaded file name

Prerequisites
  • Install esptool: pip install esptool
  • Put ESP32-P4 in download mode (hold BOOT button, press RESET)
  • Connect device via USB and identify serial port
  • Run command with appropriate serial port: python -m esptool --port /dev/ttyUSB0 ...
